## Software

Paperpile: A browser-based reference manager that downloads PDFs and saves notes and annotations to Google Drive
[[website](https://paperpile.com/features/)]

Obsidian: A powerful knowledge base on top of a local folder of plain text Markdown files
[[github](https://obsidian.md/)]

Grobid: A machine learning library for extracting information from scholarly PDFs
[[github](https://github.com/kermitt2/grobid)]

## Data

Data is Plural: A weekly newsletter of useful/curious datasets by Jeremy Singer-Vine
[[website](https://www.data-is-plural.com/)]

Awesome Public Datasets: A topic-centric list of high quality open datasets
[[github](https://github.com/awesomedata/awesome-public-datasets)]

## Visualization

[[website](https://www.data-to-viz.com/)]

MetBrewer: Color palette package in R and Python inspired by artworks at the Met by Blake Robert Mills
